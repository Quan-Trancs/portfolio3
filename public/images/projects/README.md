# Project Cover Images

Each project requires a `cover.jpg` image in its respective folder:

- `hotel_system/cover.jpg`
- `mnist_digit_classifier/cover.jpg`
- `youtube_transcripts/cover.jpg`
- `chess_game/cover.jpg`
- `whiteboard/cover.jpg`
- `collaboard/cover.jpg`
- `inference_time_agent/cover.jpg`

## Recommended Specifications:
- **Format**: JPG or PNG
- **Aspect Ratio**: 16:9 (recommended) or similar
- **Dimensions**: 1280x720px or larger
- **File Size**: Optimized for web (<500KB recommended)

You can use screenshots, mockups, or relevant images that represent each project.
