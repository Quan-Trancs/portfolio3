This is a website with a visit tracking application for Pitchfork pantry, enhancing visitor analytics with over 1000 users dataset.

## Key Features

**Visit Tracking**
- Visitor registration and check-in system
- Analytics dashboard with user insights
- Data visualization for visit patterns
- Reporting capabilities

**Data Management**
- MongoDB database for scalability
- Data collection and aggregation
- Dataset management (1000+ users)
- Performance optimization

## Technical Highlights

Developed using Next.js for server-side rendering and optimal performance. Utilizes MongoDB with Mongoose for efficient data storage and retrieval. Implements data analytics features for visitor tracking and insights, handling datasets with over 1000 users efficiently.
